Steps:

1. Clone repository
2. Run `composer install`
3. For start run `./init.sh` and go to `http://127.0.0.1:8000` url
4. For tests run `./test.sh`

Tested on `Windows 10 Enterprise` -v `21H2`

Used `Laravel 8.83.10` - doesn't require Docker to run
